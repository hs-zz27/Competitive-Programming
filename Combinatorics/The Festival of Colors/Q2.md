# 2. The Festival of Colors

### Description
During the festival, a tribe paints n totems.
Each totem must be painted one of m colors.

The tribe has one rule:
No color can appear more than k times in a row.

Count the number of valid colorings of the n totems.


### Input
- Three integers: n, m, k


### Output
Print the number of valid colorings modulo 1e9+7.

### Sample
**Input:**
```text
4 3 2
```
**Output:**
```
66
```
