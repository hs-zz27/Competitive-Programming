# 1. Riverbank Sums

**Problem**  
Emma has an array of integers representing stones along a riverbank: `a[1..n]`. She will be given `q` queries. Each query asks for the **sum** of elements in a contiguous range `[l, r]`. The array is static.

**Input**  
- First line: two integers `n q`
- Second line: `n` integers `a1 a2 ... an`
- Next `q` lines: two integers `l r`

**Output**  
For each query, print the sum of elements in `[l, r]`.

**Constraints**
- `1 ≤ n, q ≤ 2 * 10^5`
- `|ai| ≤ 10^9`

**Example**
Input
5 3
1 2 3 4 5
1 3
2 4
3 5
Output
6
9
12
